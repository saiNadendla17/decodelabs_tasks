# Project 1 - Rule-Based AI Chatbot

## Overview
This project was developed as part of the DecodeLabs Artificial Intelligence Internship.

The chatbot is built using Python and responds to predefined user inputs using if-elif-else statements.

## Features
- Greetings
- Good Morning, Afternoon, Evening and Night responses
- AI-related questions
- Programming questions
- Help menu
- Exit command
- Unknown input handling

## Technologies Used
- Python
- VS Code

## How to Run

```bash
python chatbot.py
```
